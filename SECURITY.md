# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Acorn, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: andamagodwinezra@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

I will acknowledge your report within 48 hours and work with you on a fix before any public disclosure.

## Scope

Security issues relevant to this project include:
- Credential or secret exposure
- Command injection via tool calls
- Unauthorized file access outside the working directory
- Bypassing the permission system

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.1.x   | Yes       |
| < 2.1   | No        |
